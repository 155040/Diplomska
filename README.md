За овој дипломски труд е изработена веб платформа за промовирање на скејт култура во Македонија.

Апликацијата овозможува:
  - Најава
  - Додавање, промена и бришење на скејт настани
  - Додавање, промена и бришење на скејт продавници
  - Додавање, промена и бришење на скејт спотови
  - Додавање и бришење на слики
  - Додавање и бришење на видеа
  - Приказ на скејт настани, продавници

Користени технологии:
  - Backend: Java Spring Boot
  - Frontend: React
  - Database: PostgreSQL
